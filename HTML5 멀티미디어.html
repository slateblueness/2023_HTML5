HTML 멀티미디어

video

controls: 플레이 버튼과 남은 시간, 소리 표시
autoplay: 자동실행(단, mute 상태일 때)
muted: 무음
loop: 계속 반복

플러그인

object

object태그는 다양한 파일들을 화면에 출력하는 태그

embed

embed태그는 object태그와 동일하게 사용할 수 있다
그러나 embed태그를 사용하는 것이 더 좋음

유튜브 영상 들고오기

유튜브 영상을 소스코드로 복사해 오면 iframe 안에 설정되어서 들고올 수 있다
video의 설정을 유튜브에서도 적용할 수 있다
유튜브 url 뒤에 ?를 이용한 "쿼리스트링"으로 작성하여 적용할 수 있다
쿼리스트링 형태는 ?mute=1&autoplay=1 같은 모양으로 들어간다
1은 true, 참이라는 값으로 사용 / 0은 false, 거짓이라는 값으로 사용
