관련 내용

form - input

form: input태그의 값을 서버로 전달하는 전체태그

input: 값을 입력 받는 태그

input태그를 이용해서 값을 전달할 때 name과 value값을 통해서 전달

submit: 내용을 form을 통해 제출
button: 클릭할 수 있는 공간

태그들간의 관계

하나만 사용한 태그

동일한 위치에 있으면 형제태그(앞 공백이 같은 라인)

div의 자식 태그가 되는 p태그(자식 태그의 앞 공백이 한 칸 더 들어감)

p태그의 부모 태그가 되는 div태그

앞 공백 질서가 잘 지켜지지 않은 경우 fomat document 기능을 통해 정렬 가능
